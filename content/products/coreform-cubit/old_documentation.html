Coreform Cubit license activation and license update.

Instructions for activating new licenses and updating existing licenses. Coreform products require activation using a unique product key. Products that have already been activated but have been upgraded to a new version or that have changed need to be updated to reflect the changes.

RLM-Server installation instructions.

For network installations, RLM-Server software manages floating Coreform Cubit licenses. RLM-Server is installed and activated on your designated license server.
