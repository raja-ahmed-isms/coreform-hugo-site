Software maintenance

Stay up to date with the Coreform software maintenance program.

Coreform is continually improving its software in order to provide you with the latest technology and enhanced features that will make meshing your models easier. The Coreform Software Maintenance Program ensures that you are getting the very best software available.

Maintenance benefits.

The Software Maintenance Program provides the following benefits during the maintenance period:

  • Receive the latest software updates, patches, and new releases
  • Standard technical support package (See Standard Technical Support)

Maintenance for lease licenses.

No additional maintenance cost is required for customers with lease licenses. Lease licenses include software updates and Standard Technical Support for the length of the lease.

Maintenance for perpetual-use licenses.

Customers who purchase a perpetual-use (paid-up) license of Coreform software receive a license to use the then released version of the software in perpetuity and Standard Technical Support for the first year. After the first year, customers can get new releases, software updates, and continued Standard Technical Support by paying an annual maintenance fee. See Maintenance Cost below.

Coreform will contact you prior to the end of your maintenance period to help you renew your software maintenance.

Maintenance cost.

Software maintenance is included with your annual lease. For customers with perpetual-use licenses, the Software Maintenance Program is included for the first year and is available for an additional annual fee for subsequent years. Contact Coreform for a current maintenance information.

Lapsed maintenance.

Coreform customers who purchase a perpetual-use license will renew their maintenance each year. If the software maintenance expires, updates and support are no longer available. To renew lapsed maintenance, you will need to pay the maintenance fee for the period during which your maintenance was expired, in addition to the current annual maintenance fee.

Contact Coreform for more information about our software maintenance program.
