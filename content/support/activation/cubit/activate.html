Activating Coreform Cubit


Product license keys are used to activate Coreform software. Below are instructions for activating node-locked and floating licenses. Activation of Coreform software can be done online or, if the computer is not connected to the Internet, by downloading your unique license file.

  • Coreform Cubit Learn activation
  • Node-locked activation (standalone computer)
  • Floating activation (network installation)
  • Updating existing licenses
  • Offline activation (without Internet connection)
  • Data translator activation
  • Transfer/Deactivate a license

If you get an error message during installation, please see Coreform Cubit FAQs for possible solutions. If you have any questions about activating your software, please contact Coreform support.
