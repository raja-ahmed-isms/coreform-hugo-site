Offline activation

Activating a license when not connected to the Internet.

Product license keys are used to activate Coreform software. Below are instructions for activating licenses when the computer you are activating is not connected to the Internet.

If the computer running Coreform Cubit is not connected to the Internet, you will need to use the product key to activate and download the license file using another computer that can access the Internet.

To manually activate your software:

  1. Install and launch Coreform Cubit.
  2. Obtain the hostid of your computer using the rlmutil utility. Navigate to the installation folder in a terminal window. Type rlmutil rlmhostid ether



  3. On a device with Internet access, select the license in your account's My License area.
  4. At the bottom of the page, click Activate.
  5. Enter the hostname and hostid of the computer, then click Add Activation.



  6. Scroll to Activation at the bottom of the license page and click on the download icon next to the license.



  7. Save the node-locked license file to your Cubit X.X/bin/licenses folder.

    Use your product key as the filename, for example AL218C0-6M50F4-E46371-5313ED-8E8B125.lic By default, the folder is located here:
    • Windows: C:\Program Files\Cubit 16.X\bin\licenses\
    • Linux: /opt/Cubit-16.X/bin/licenses/
    • Macintosh: /Applications/Cubit-16.X.apps/MacOS/licenses/

Note: A node-locked product key will activate Coreform Cubit on a single computer. A floating product key will activate a floating license of Coreform Cubit on the RLM-Server license server.

If you get an error message during installation, please see Coreform Cubit FAQs for possible solutions. If you have any questions about activating your software, please contact Coreform support.
