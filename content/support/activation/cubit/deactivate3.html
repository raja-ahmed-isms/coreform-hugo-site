License deactivation

Transfer or deactivate a license

Licenses can be transferred from one computer or server to another.

  • Transfer a node-locked license
  • Transfer a floating licenses

Transfer a node-locked license

A node-locked license is installed on a single standalone computer. Even if the computer is connected to a network, the license is tied to the standalone machine. As long as the license is valid, the license can be deactivated and transferred to another standalone computer.

License deactivation

To transfer a node-locked license to a different computer, first deactivate the license on the original computer.

  1. Open the Product Activation window on the Help menu.

  2. Enter the product key of the license and click the Deactivate button.

    NOTE: The product key can be found on the My Licenses page of your Coreform account.
  3. The deactivation warning will appear asking you to confirm the deactivation.

  4. When the license is deactivated, press OK to close the app.

  5. Uninstall the software from the old computer. You may need to remove the installation folder to completely remove the software.
  6. Install the software on the new computer. During the installation process you will enter a product key. Use the original product key to activate the license on the new computer.

Deactivation failed

Deactivation number exceeded

By default, each license has one deactivation allowing you to make one transfer. If you receive a message that the deactivation failed because the number of deactivations has been exceeded, please contact support@coreform.com to request additional deactivations. There is no charge for additional deactivations.

File not found

A license file usually is named using the product key number with a .lic extension (for example, FZBC2CD-3128C4-20F964-DEC545-0300CE4.lic). If you activated the license from your license page or if you changed the file name, you should change the filename to the productkey.lic format and try to deactivate again.

NOTE: The license file can be found in the /Cubit XX.X/bin/licenses folder.

Transfer a floating license

A floating license is installed on a network server and shared by computers over the network. The license is tied to the server. As long as the license is valid, the license can be deactivated and transferred to another server.

License deactivation on the server

To transfer a floating license to a different server, first deactivate the license on the original server before activating the license on the new server.

Windows
  1. Run the rlm_activate utility to open the Product Activation dialog. This can be done from the Start menu or by double-clicking "rlm_activate.exe" in the "C:\Program Files\RLM-Server" folder.
  2. On the RLM Activation dialog, enter the product key and press the Deactivate button. The Hostname should fill in automatically.



    If not successful, see Deactivation Failed below for help.
  3. Uninstall RLM-Server, then delete the RLM-Server folder to remove any remaining log files.
  4. Download the new RLM-Server from Downloads and install the RLM-Server software on the new server.
  5. Activate the floating license on the server using the original product key. For more information, see RLM-Server installation instructions.
Linux, Mac OS X, or Windows Command Prompt
  1. Run from a terminal as administrator: "rlm_activate --deactivate <filename.lic> <product key>" where <filename.lic > is the name of the license file and <product key> is product key for the license.





  2. Uninstall RLM-Server, then delete the RLM-Server folder to remove any remaining log files
  3. Download the new RLM-Server from Downloads and install the RLM-Server software on the new server.
  4. Activate the floating license on the server using the original product key. For more information, see RLM-Server installation instructions.
