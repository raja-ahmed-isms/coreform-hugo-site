Floating license activation

Activating a floating license.

Product license keys are used to activate Coreform software. Below are instructions for activating floating licenses. Activation of Coreform software can be done online or, if the computer is not connected to the Internet, by downloading a unique license file.

To setup a floating license on a network, you will

  • install RLM-Server software on a license server and activate the license
  • install Coreform Cubit on individual network computers and indicate the license server IP address

Already have RLM on your server?

If you already are using RLM on your server, you can simply use the csimsoft.set file and your license file.

Install and activate RLM-Server.

To activate a floating license of Coreform Cubit on a network:

  1. Download RLM-Server and install it on the computer that will be the network license server.
  2. Open a command prompt or terminal window and navigate to the RLM-Server folder.
  3. Run rlm_activate to open the RLM Activation window



    or

    From a terminal run as administrator: ".\rlm_activate --activate <product_key> <hostname>" where <product_key> is the product key for the license and <hostname> is the name of the server . The hostname is optional, but will identify the server in your list of licenses in your account's My Licenses area.



    If successful, you will see the following message:



  4. You may need to restart the server to recognize the new license file. In a web browser, type " http://localhost:5054" in the address bar to access the Reprise License Server Administration page. Click Reread/Restart Servers from the left sidebar. Check that the Coreform ISV server is running by clicking Status. Coreform should appear in the ISV Server table.



    See RLM-Server Installation Instructions for more information.

Install Coreform Cubit.

To setup Coreform Cubit on a network computer:

  1. Download Coreform Cubit and install it on each network computer that will run Cubit.
  2. Launch Coreform Cubit.
  3. In the Product Activation window, select Floating License: Connect to a license server.
  4. Enter the hostname or IP address of the license server where RLM-Server is located. The default port number is 5053. See RLM-Server Installation Instructions for information about changing the port.



  5. Click Activate

    If successful, you will see the following message:



Note: Network installation requires a floating license of Coreform Cubit. A network product key is used to activate the RLM-Server software. Do not use the network product key to activate Coreform Cubit on the client computer.

If you get an error message during installation, please see Coreform Cubit FAQs for possible solutions. If you have any questions about activating your software, please contact Coreform support.
