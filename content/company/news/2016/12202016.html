AMERICAN FORK, UTAH, December 20, 2016 — csimsoft today announced the completion of an IMPETUS Afea solver exporter that supports Hex64 elements. The cubic elements can be limited to just the elements on the model boundary or throughout the model. The support for cubic elements allows IMPETUS to run a more precise analysis on model surfaces.

"csimsoft is pleased to partner with CertaSIM to support the IMPETUS Afea solver. IMPETUS is an extremely accurate explicit dynamic solver," said Paul Ressler, csimsoft Director of Sales and Marketing. "We support HEX64 elements specifically for IMPETUS. Now, engineers can create meshes that can be solved with IMPETUS on a GPU in a fraction of the time it takes other solvers."

For more information about the IMPETUS export, contact CertaSIM (www.certasim.com)

About csimsoft

csimsoft is a leading developer of simulation tools for generating high-quality meshes for challenging computational fluid dynamics and finite element analysis simulations. csimsoft was founded in 2004 to provide mesh generation tools for challenging simulations.
